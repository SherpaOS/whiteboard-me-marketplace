# Whiteboard-Me — Public Marketplace

Make the viral "whiteboard sketch of my work life" graphic — without the AI-image problems. `whiteboard-me` builds a **research-grounded work-life map**: what you do, who you work with, your values, tools, and what you're building — rendered as crisp HTML/SVG, so every word is correct and editable. No hallucinated bios, no mangled text.

## Install

In Claude Code / Cowork:

```
/plugin marketplace add SherpaOS/whiteboard-me-marketplace
/plugin install whiteboard-me@whiteboard-me
```

Then ask Claude: *"make me one of those LinkedIn whiteboard sketch graphics"* or *"visualize my work life"*.

## What you get

- **Grounded research** — every claim on the map traces to a real source (your profile, connected systems, LinkedIn, or you). An approval gate before rendering.
- **Text fidelity** — HTML/SVG canvas (1600×2000, LinkedIn-friendly 4:5), not an AI image. Crisp, correct, editable.
- **Deterministic ink-sketch portrait** — your actual face, cross-hatched line art, no likeness drift. Free and offline.
- **Variants** — square, story, and a deliberately-incomplete Instagram teaser that points to the full map on LinkedIn.
- **Business mode** — map a company on one canvas with the same engine.
